# Skin Cancer Detection App
App to detect skin cancer.

## Features

- **Skin Cancer Detection**
- **Prediction History**
- **Cancer News** 

## Tools

- **TFLite Model**
- **Save history in Room** 
- **uCrop for Image Cropping**
- **ViewPager2 for Cancer News View**
